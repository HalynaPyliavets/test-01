Cписок команд, які ми розглядали при роботі з терміналом на лекції від 06.09.22:

  • git clone (клонування репозиторію)
  • git status (інформація в якому стані файли)
  • cd test-01 (перехід в папку, яка контролюється Git, де test-01 це назва папки)
  • git add . (додавання файлів у закладку "підотовка до відправки", крапка в кінці команди add фіксує всі нові документи в списку змін за один раз)
  • git commit -m "feat: initial project" (створення коміту - повідомлення про зміни, де "feat: initial project" - це назва коміту)
  • git push (публікація змін в Git)
  • git сheckout (для відміни змін)
  • gitt pull (стягування останніх даних з усіх документів)
  • git merge (об'єднання гілок в одну)
  • git log (переляд історії проекту (комітів))
  • git branch (список гілок)
